1. Tag Name h1

    It is the highest section level of heading.

  2. Tag Name br

    It is the element produces a line break in text. For example, a
    element is included at each point where we want the text to break. The text after the
    begins again at the start of the next line of the text block.

  3. Tag Name hr

    This element represents a thematic break between paragraph-level elements: for example, a change of scene in a story, or a shift of topic within a section.

  4. Tag Name super scripted

    This element specifies inline text which is to be displayed as superscript for solely typographical reasons.Superscripts are usually rendered with a raised baseline using smaller text.

  5. Tag Name sub scripted

    This element specifies inline text which should be displayed as subscript for solely typographical reasons.Subscripts are typically rendered with a lowered baseline using smaller text.

  6. Tag Name underlined

    This element represents a span of inline text which should be rendered in a way that indicates that it has a non-textual annotation.

  7. Tag Name small

    This element represents side-comments and small print, like copyright and legal text, independent of its styled presentation. By default, it renders text within it one font-size smaller, such as from small

  8. Tag Name mark

    The element represents text which is marked or highlighted for reference or notation purposes due to the marked passage's relevance in the enclosing context.

  9. Tag Name li

    This element is used to represent an item in a list. It must be contained in a parent element: an ordered list (ol), an unordered list (ul), or a menu (menu). In menus and unordered lists, list items are usually displayed using bullet points. In ordered lists, they are usually displayed with an ascending counter on the left, such as a number or letter.

  10. Tag Name marquee

    This element is used to insert a scrolling area of text. We can control what happens when the text reaches the edges of its content area using its attributes.
